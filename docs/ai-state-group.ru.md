# Группа состояний AI

- Последнее обновление: 2026-04-23
- Имя группы: `ai`
- Команда: `just group ai`
- Время выполнения: ~0.4 секунды

## Включённые состояния

| Состояние | Назначение |
|-----------|------------|
| `ollama` | Локальный LLM‑сервер (Ollama) |
| `llama_embed` | Модели эмбеддингов для RAG |
| `nanoclaw` | Навигация по коду и анализ |
| `opencode` | AI‑ассистированная разработка (OpenCode) |
| `image_gen` | Генерация изображений (Stable Diffusion, Flux) — по feature flags |

## Управление секретами

Секреты, связанные с AI (API‑ключи, токены доступа), хранятся в `gopass` с бэкендом `age`/`Yubikey`.

- `ai/ollama/api_key` (если требуется)
- `ai/opencode/...`
- `ai/image_gen/...`

## Применение группы

```bash
just group ai
```

Эта команда применяет все AI‑состояния в правильном порядке.

## Добавление новых AI‑состояний

1. Создайте состояние в `states/` (например, `states/new_ai.sls`)
2. Добавьте его в `states/group/ai.sls`
3. Обновите этот документ
4. Убедитесь, что все секреты интегрированы с `gopass`
